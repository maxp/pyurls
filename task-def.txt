
URL Shortener (сокращатель ссылок)

Стек: Python + FastAPI + SQLite + Docker

Задача: Создать веб-сервис для сокращения URL-адресов (аналог bit.ly).

Функционал:

  Сокращение ссылки
    Эндпоинт: POST /api/v1/shorten
    Принимает URL, возвращает короткую ссылку с уникальным кодом.
  Редирект
    Эндпоинт: GET /s/{code}
    Выполняет редирект на оригинальный URL и увеличивает счётчик переходов.
  Статистика
    Эндпоинт: GET /api/v1/stats/{code}
    Возвращает оригинальный URL, количество переходов и дату создания.

Требования:

  FastAPI — с автоматической документацией (Swagger — /docs).
  SQLite — хранение ссылок и счётчика переходов.
  Валидация URL — возвращать 400 при неверном формате.
  Уникальный 6-символьный код ([a-zA-Z0-9]).
  Docker — образ на основе python: 3.11, запуск через docker-compose up.
  requirements.txt — зависимости.
